For this assignment MATLAB was used.

The MDPToolbox for MATLAB is the library used to calculate policy and value iteration approaches.

The MDPToolbox can be installed in a variety of languages (including R, GNU Octave and Python), with instructions located here: http://www7.inra.fr/mia/T/MDPtoolbox/

The problems used are Car Race example which can be downloaded here: https://mulcyber.toulouse.inra.fr/frs/?group_id=11&release_id=579

And a grid world example which was shown in the lectures. 


---


To create some of the plots R was used. To run these plots, please install `ggplot2`, `reshape2`, and `gridExtra` packages.
